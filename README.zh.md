# Quick Date Picker

[English](README.md) | [中文](README.zh.md)

在 Obsidian 文档中快速输入日期的插件。输入 `@`（或自定义触发字符）即可呼出迷你日历，选择日期并一键插入。

## 功能特性

- **迷你日历弹窗**：输入触发字符后实时弹出日历，支持鼠标点选或键盘导航
- **相对日期快捷输入**：支持 `@+3d`、`-1w`、`2m` 等快捷语法，无需打开日历
- **多种输出格式**：内置多种日期格式（标准、Wiki 链接、中文、紧凑等），支持自定义前缀/后缀
- **格式即时切换**：在迷你日历弹窗底部直接显示常用格式按钮，无需进入设置即可快速切换输出格式
- **标题栏支持**：在笔记标题（文件名）中同样可以使用日期选择器
- **高度可定制**：自定义触发字符、格式模板、记住上次使用的格式

## 安装

### 从 Obsidian 社区插件市场安装（推荐）

1. 打开 Obsidian，进入 **设置 → 社区插件**
2. 关闭安全模式（如果尚未关闭）
3. 点击 **浏览**，搜索 "Quick Date Picker"
4. 点击安装并启用

### 手动安装

1. 从 [Releases](https://github.com/ritalee2333/obsidian-quick-date-picker/releases) 下载最新版本的 `main.js`、`manifest.json` 和 `styles.css`
2. 将文件放入你的 Obsidian 仓库的 `.obsidian/plugins/quick-date-picker/` 目录
3. 重启 Obsidian，在设置中启用插件

## 使用方法

### 在文档正文中使用

1. 在编辑器中输入 `@`（或你设置的触发字符）
2. 弹出迷你日历，使用以下方式选择日期：
   - **鼠标**：点击日期单元格选中（可选点击格式按钮切换格式），然后按 Enter 或点击「确定」按钮确认插入
   - **键盘**：方向键移动，Enter 确认，Escape 关闭
   - **相对日期**：直接输入 `+3d`、`-1w`、`2m`、`-1y` 等自动转换为对应日期
3. 确认后日期文本自动插入到光标位置

### 在标题中使用

1. 点击笔记标题进入编辑状态
2. 输入触发字符，同样会弹出日历
3. 选择日期后自动替换触发文本

### 相对日期语法

| 输入 | 含义 |
|------|------|
| `+3d` | 3 天后 |
| `-1w` | 1 周前 |
| `2m` | 2 个月后 |
| `-1y` | 1 年前 |

单位说明：`d`=天, `w`=周, `m`=月, `y`=年。`+` 可省略。

## 设置

进入 **设置 → 社区插件 → Quick Date Picker** 进行配置：

- **触发字符**：自定义呼出日历的字符（默认 `@`）
- **记住上次使用的格式**：开启后下次弹窗自动选中上次使用的格式
- **默认格式**：设置默认的日期输出格式
- **常用格式列表**：添加、删除、排序常用的日期格式模板

### 格式模板语法

支持以下日期标记：

| 标记 | 说明 | 示例 |
|------|------|------|
| `YYYY` | 四位年份 | 2026 |
| `YY` | 两位年份 | 26 |
| `MM` | 两位月份 | 05 |
| `M` | 一位月份 | 5 |
| `DD` | 两位日期 | 23 |
| `D` | 一位日期 | 23 |

前缀/后缀示例：`前缀 [[` + `YYYY-MM-DD` + `后缀 ]]` = `[[2026-05-23]]`
